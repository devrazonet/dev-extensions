Extensões para simplificar o dia a dia no BackOffice.
Como utilizar? Baixe a extensão correspondente ao seu browser (baixar aqui)
Se for arquivo .ZIP, descompacte em algum lugar seguro do seu computador
Acesse o Menu (ou ícone) Extensões - Gerenciar extensões - Carregar sem compactação.
Pine a extensão para ficar sempre visível no seu browser
No página de notificações do back, clique sobre a extensão e depois no botão.
Se houver notificações não finalizadas, a extensão entrará em ação.

Para Chrome e relacionados, é preciso ativar a opção Modo desenvolvedor para poder carregar as extensões. No Opera não é necessário.
   Para Firefox *    Finalizar todas as notificações    Botão Ticket no menu principal    Execute as extensões direto no browser Firefox.
   Para Chrome / Opera / Edge *    Finalizar todas as notificações    Botão Ticket no menu principal    Baixe o .zip e importe no gerenciador de extensões do browser. Compatível também com outros browsers de base Chromium.
